A Custom Frame

A simple data display window, consists of
  • A Menubar with most frequenty used items
  • A Toolbar with a few essential buttons
  • A status area
  • A Message logger and a File open/save dialog
The frame is supposed to act as a base class for more sophisticated custom frames. This window provides a consistent look and feel for all the windows across the application. It also provides a uniform strtegy for closing a window. It however does not dictate placement of the components beyond a minimum.

In order to implement a help window for the derived application window, all one has to do is invoke setHelpFile() method and pass the appropriate filename and the help window size.
