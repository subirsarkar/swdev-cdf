SVT Word Decoder

Decodes hex words into relevant physics quantities and displays as a table.
  • End Event Words
  • Hit words
  • Roads
  • Road-Hit Packets
  • Svt Tracks
  • XFT Tracks

The window consists of an input area, the word type options, an output area and a set of buttons. Data can be copied onto the input area from another window in the same application. multi-word line is properly split. Hex words may or may not contain '0x'. If the number of digits in a word is > 6, that word is discarded. One cannot copy data from a native applicatio (terminal) and paste onto the input are. In future reading data from an input file will be enabled.
