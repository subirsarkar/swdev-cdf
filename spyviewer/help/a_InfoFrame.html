Individual Spy Buffer Display Window

Displays individual Spy Buffer informations in a number of ways. The window consists of
  1. The MenuBar
    • File; Connect to, Disconnect from the SmartSockets server; usual basic services.
    • Options; When run online the display automatically gets updated with most recent data. Auto update of display can be switched on/off. Turning Debug option on is expert only.
    • View; one can view (1) all the words, (2) EoE words alone. For Physics format, the first option does not show the decoded EE but the raw data display contains all the words
    • Action; (1) Open a tiny tool to decode SVT words into physics format interactively, (2) Analyse the Spy buffer dump. This is the most important analysis-level feature of the tool.
  2. The ToolBar; when used in standalone mode, contains Connect and Disconnect buttons to establish/stop communication with the SmartSockets Server
  3. A Tree showing all the Crate->Board->SpyBuffers. The list is dynamically built from crate status messages. This list allows us to browse all the buffers from a single window. As soon as the first set of Svt Status messages is received from all the crates, the application unsubscribes for Crate Status message. The tree shows the list of all the buffers of the SVT system but by default only a few buffers are sent from the crates as messages. We distinguish those buffers which contain data from others which do not by changing the icons as soon as data is received for that buffer.
  4. Spy Buffer Dispplay panel. The panel consists of,
    • Global Error (EE) status, which consists of
      • Spy Buffer status window,
        • Spy pointer value
        • Freeze status bit
        • Pointer overflow bit
      • Most recent status of the 8 individual EE Error bits
      • EE error informations in some more detail in tabular form. The table (at the bottom of Global Error window) displays
        • Number of times EE errors have occurred and the % fraction of occurrance of EE errors in the latest buffer.
        • Total cumulative number of times EE errors have occurred and the % fraction of occurrance of EE errors.
    • Buffer display, (1) Board format, (2) Raw Hexadecimal format.
    • A plotting widget, consists of a canvas and a list. For each type of buffer a few basic quatities are histogrammed. There is an experimental interactive histogramming panel also (The 'Command' tab next to the 'List' tab which displays histograms which are craeted and filled automatically.)
  5. The Message Logger area, display SmartSockets related messages etc.
  6. The StatusBar; displays,
    • Status messages like 'Ready', 'Updating', Tooltip on buttons, menu etc.
    • Fully qualified name of the buffer
    • Total number of words received for that buffer
    • 'All' or 'EE' words
    • Available data formats
