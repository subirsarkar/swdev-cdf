spyGUI 1.0

User interface for SVT Spy Buffer Monitoring and analysis (SPYMON), which is used to monitor the CDF Silicon Vertex Trigger performance in real time. spyGUI
  • Displays SVT hardware and analysis errors in detail,
  • Works as an Online Beam Position Monitor,
  • Works as a Histogram Producer and Viewer,
  • Configures analysis options,
  • Works as an SVT Database Browser.
