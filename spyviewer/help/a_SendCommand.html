Sending Application Specific Commands

Send application specific commands to the selected destination(s) using predefined commands,
  • Start, Stop, Suspend, Resume spymon process in the selected crate(s)
  • Reset histogram content
  • Reset Board Register Errors
The Stop command can be used to stop the consumer processes as well. In future we may add a free text input area used to send the command name.

One can also send configuration messages from a file, either sending
  • simply the Filename, or
  • the file contents as strings
Sending configuration from file could be useful when the same configuration properties are altered time and again. However, sending an arbitrary file may create trouble for the crate processes.
